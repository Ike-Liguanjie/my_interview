# Python
## 1 深浅拷贝
深拷贝不用多说，完全新建一个内存地址将内容拷贝出来，修改不会影响到原来的数据，这里主要讲一下浅拷贝。<br>
在浅拷贝时候要注意，原对象中的可变元素是拷贝引用，修改会影响原有数据！<br>
什么意思呢，举个简单例子：<br>
```
a = (1, [1, 2], {1, 2}, 4)
b = list(a)
b[1].append(3)
b[2].add(3)
b.append(5)
print(a)
print(b)
```
这个时候的打印结果会是:<br>
```
(1, [1, 2, 3], {1, 2, 3}, 4)
[1, [1, 2, 3], {1, 2, 3}, 4, 5]
```
可以发现在单纯对b进行append的时候，a是不会发生改变的，但是对里面的list和set做操作时，原有数据就会发生改变。<br>
注意：list()方法和列表生成器、字典生成器都属于浅拷贝<br>
